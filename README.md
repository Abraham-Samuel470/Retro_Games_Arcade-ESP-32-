# Retro_Games_Arcade-ESP-32

# 🎮 ESP32 Retro Gaming Console

This is a pocket-sized **retro gaming console** built using the **ESP32** microcontroller, featuring an OLED display, push-button controls, and buzzer for audio effects. Play classic-inspired games like Dino Run, Flappy Bird, Pong, Space Shooter, Maze Escape, and Brick Breaker — all from a clean game menu.


## 🧠 Features

- ✅ OLED Display (128x64) UI
- ✅ Buzzer sound effects for gameplay
- ✅ Physical buttons for control
- ✅ Game selection menu
- ✅ Score display & level system
- ✅ Compact and fun retro aesthetics

---

## 🕹️ Games Included

1. **Dino Run** – Jump over cactus obstacles!
2. **Flappy Bird** – Navigate through the pipes.
3. **Pong** – Classic paddle vs paddle game.
4. **Space Shooter** – Shoot down incoming enemies.
5. **Maze Escape** – Navigate the maze and find the exit.
6. **Brick Breaker** – Smash the bricks with a bouncing ball.

---

## 🛠️ Hardware Used

| Component         | Description                       |
|------------------|-----------------------------------|
| ESP32             | Main controller                   |
| OLED Display      | 128x64 I2C SSD1306                |
| Push Buttons      | Game controls (up/down/select/etc)|
| Buzzer            | Sound effects                     |
| Jumper Wires      | Connections                       |
| Breadboard/PCB    | For mounting components           |

---

## 🔌 Pin Connections (Example)

| Component       | ESP32 Pin |
|----------------|-----------|
| OLED SCL       | GPIO 22   |
| OLED SDA       | GPIO 21   |
| Button 1       | GPIO 14   |
| Button 2       | GPIO 27   |
| Button 3       | GPIO 26   |
| Button 4       | GPIO 25   |
| Buzzer         | GPIO 4    |
